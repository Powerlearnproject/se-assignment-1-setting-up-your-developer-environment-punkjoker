Windows 10 Download the Installer: Go to the Visual Studio Code download page. Click on the Windows download button. Run the Installer: Locate the downloaded file. Double-click the installer to run it. Setup Wizard: Follow the setup wizard. Accept the license agreement. Choose the destination folder for the installation. Select the additional tasks you want, such as creating a desktop icon, adding to PATH, and associating file types with VS Code. Click "Next" and then "Install." Launch VS Code: Once the installation is complete, you can launch VS Code from the Start menu or the desktop shortcut. Example of git repo: ## https://github.com/punkjoker/Learning-python.git
